这里面的matlab代码原代码和一些数据文件
*******************************************************************
其中带有orla_*文件是第二个算法的一些数据文件和源代码
/orla_data/     运行过程中数据文件保存在这个文件下
/orla_dataRij/  JAVA code执行之后会产生一些数据文件,放在这个文件下，Rij*.txt中提供的数据是迭代时间和对应的车辆对应基站的Rij，以及对应基站的连接数目
/orla_image/    matlab 数据统计之后的一些图片信息
/orla_resource/ 统计数据的一些matlab源代码
        ../drawdealRij.m  这个数读取java code的运行文件，获取统计数据，并且画累积分布图
        ../aboutnumtime.m 统计每次迭代车辆的Rij和运行时间
        ../orla_distance.m 重新计算需要参见distance数据文件

*******************************************************************
第二次运行的matlab数据有一部分来自，第一次算法已经处理的matlab数据

*******************************************************************
第一次算法的matlab
/data/      表示matlab处理之后保存的一下数据文件
/dataRij/   java code 算法执行后需要统计的一下文件Rij数据，基站上连接的数据，迭代次数，以及其他迭代时间数据
/disatance/ matlab 处理数据信息，获取车辆到基站的距离，其中没有的车辆的距离信息我这里使用 -1 这个数值，因为距离不可能是负数
/image/     将第一个算法的一些情况的图形保存在这
/resource/  数据处理的一些源代码的实现，其功能在代码中都有注释的

/final_data/ 该文件是保存的是原始的待处理的数据信息

*******************************************************************
